# Jinshucraft

为服务整合包Karma2而产生的一个大型的Minecraft扩展mod，需要安装customstuff2，blockproperties以及minetweaker等。

关于玩法的讨论帖在百度贴吧[mc工业2吧](https://tieba.baidu.com/p/4996517453?see_lz=1)上发布。
